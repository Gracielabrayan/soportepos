# 🖥️ Soporte POS - GDN Argentina

Sitio web informativo y de gestión interna para el equipo de Soporte N2 especializado en sistemas NCR.

### 📋 Descripción
Este proyecto centraliza el acceso a herramientas de diagnóstico, reportes de incidentes (Jira) y documentación técnica para optimizar la operatividad en las sucursales.

### ⚙️ Características Técnicas
* **Arquitectura:** Estructura de activos segregada (Assets/CSS/JS).
* **UI/UX:** Diseño con soporte para Modo Oscuro/Claro.
* **Responsive:** Adaptado para dispositivos móviles (útil para técnicos en campo).
